Класс находится в файле LineCassini.h, в файле LineCassine.cpp находятся его методы.
В папке lib находится собранная статическая библиотека содержащая методы класса.
Для модульного тестирования используется код из testing.cpp.
Для ручного тестирования используется консольное приложение, в файле main.cpp находится меню приложения, а в menu.cpp код фукции, которые используются в main.cpp.
